\section{Грамматика}
\subsection{Структура китайского предложения}
\large
Для китайского языка характерен фиксированный порядок слов.\\
Простое, наиболее употребительное предложение выстраивается по формуле\\
\begin{center}
	“подлежащее + сказуемое + дополнение”
\end{center}
Остальные члены предложения размещаются внутри этой структуры.
\begin{example} \hfill \break
	\LARGE
	我爱妈妈。\\
	我喝水。
\end{example}
При построении предложения с качественным сказуемым как правило используется 很.
\begin{example}\hfill \break
	我很好。\\
	他们很可爱。
\end{example}
При построении общего опроса (с ответом да/нет) используется вопросительная частица 吗. Если в предложении качественное сказуемое, 很 не используется.
