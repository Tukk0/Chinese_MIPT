\section{Грамматика}
\subsection{Структура китайского предложения}
\large
Для китайского языка характерен фиксированный порядок слов.\\
Простое, наиболее употребительное предложение выстраивается по формуле\\
\begin{center}
	“подлежащее + сказуемое + дополнение”
\end{center}
Остальные члены предложения размещаются внутри этой структуры.
\begin{example}
	Test
\end{example}

\end{table}
