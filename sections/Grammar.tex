\section{Грамматика}
\subsection{Структура китайского предложения}
\large
Для китайского языка характерен фиксированный порядок слов. Простое, наиболее
употребительное предложение выстраивается по формуле “подлежащее + сказуемое
+ дополнение”. Остальные члены предложения размещаются внутри этой структуры.
