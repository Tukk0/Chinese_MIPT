\subsection{Занятие 19. 2024.11.19}
\textbf{Изменения тона 一}\\
Перед слогом с первым, вторым или третьим тоном 一 произносится четвертым тоном.\\
Перед слогом с четвертым тоном 一 произносится вторым тоном.
\begin{newgraphems}
	\item 法 - fǎ - морфема "закон, правило"
	\item 字 - zì - письменный знак
	\item 寸 - cùn - вершок (мера длины др. Китая = 3.33 см)
	\item 弓 - gōng - лук
	\item 长 - cháng - длинный / zháng - расти
\end{newgraphems}
\begin{newwords}
	\begin{enumerate}
		\item 一点 - yīdiǎn - немного
		\item 一点儿 - yīdiǎr - немного
		\item 一点点 - yīdiǎndiǎn - чуть-чуть (совсем немного)
		\begin{example}\hfill\break
			我会说一点汉语。\\
			我会说一点儿汉语。\\
			我会说一点点汉语。
		\end{example}
		\item 点 - diǎn - точка
		\item 有点儿 - yǒudiǎr - немного\\
		有点儿 используется в нежелательном для говорящего смысле:
		\begin{example}\hfill\break
			有点儿大 - великоват\\
			有点儿短 - коротковат
		\end{example}
		\item 短 - duǎn - короткий
		\item 语法 - yǔfǎ - грамматика
		\item 汉字 - hànzì - иероглифы
		\item 见 - jiàn - видеть
		\item 觉得 - juéde - думать, считать, полагать
		\item 土 - tǔ - почва
		\item 在 - zài - 1) находиться, быть\\
		2) предлог нахождения "в"\\
		Предпочитетельный порядок слов: КТО-ГДЕ-ЧТО
		\begin{example}\hfill\break
			我在这儿。 - Я здесь.\\
			我在这儿。 他在哪儿。 - Я здесь, мой друг там.\\
			我在大学学习中文。 - Я учу китайский в университете.\\
			他不在教学楼。 Он не в учебном корпусе.\\
		\end{example}
		\item 在哪儿 - где (находится)
		\begin{example}\hfill\break
			你在哪儿? - ты где?\\
			大学在哪儿 - где университет?\\
			你来自哪里？ - Откуда ты приехал?
		\end{example}
		\item 可是 - kěshì - союз "а, но"
		\item 法国 - fǎguó - Франция
		\item 近 - jìn - близко
		\item 最近 - zuìjìn - в последнее время
		\item 早 - zǎo - рано, ранний
		\item 早上 - zǎoshang - утро
		\item 早上好 - zǎoshanghǎo - Доброе утро!
		\item 晚 - wǎn - поздно, поздний
		\item 晚上 - wǎnshang - вечер (вечером)
		\item 晚上好 - wǎnshànghǎo - Добрый вечер!
	\end{enumerate}
\end{newwords}
Отличие 学 и 学习\\
学 употребляется всегда с дополнением. 我学汉语。\\
学习 может употребляться и само по себе.
