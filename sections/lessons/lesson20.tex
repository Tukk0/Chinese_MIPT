\subsection{Занятие 20. 2024.11.21}
Слова, обозначающие степень признака
\begin{itemize}
	\item 特别 - tèbié - особенно (очень)
	\begin{example}\hfill\break
		汉语特别难。 - Китайский язык особенно трудный.
	\end{example}
	\item 好 + verb - положительно ... (трансформация глагола в прилагательное)
	\begin{example}\hfill\break
		好喝 - вкусный по питьевым ощущениям\\
		中国茶很特别好喝 - Китайский чай очень вкусный.\\
		好吃 - вкусный (о еде, 吃 - chī - кушать)\\
		好学 - легко дается в плане учебы
	\end{example}
	\item 听 - tīng - слушать
	\begin{example}\hfill\break
		好听 - приятный на слух, благозвучный
	\end{example}
	\item 难 + verb - трудно ... (трансформация глагола в прилагательное)
	\begin{example}\hfill\break
		难学 - труден в изучении
	\end{example}
\end{itemize}
